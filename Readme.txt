O aplicativo SaborExpress serve para registrar restaurantes e ativar ou desativar o cadastro dos restaurantes. 
Além disso, ele também exibe restaurantes registrados, porém não está salvando ainda os arquivos.

Este foi meu primeiro projeto desenvolvido em Python, entre Setembro e Outubro de 2024.

Para executar, basta abrir o arquivo app.py em um aplicativo como o Visual Studio Code.
